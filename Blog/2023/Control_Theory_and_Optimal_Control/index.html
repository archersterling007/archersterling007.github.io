Optimal Control

Main References:

  • An extensive reference for Dynamic Programming is Dynamic Programming and Optimal Control by Dimitri P. Bertsekas - Website
  • Classical references for optimal control:
    • Applied optimal control : optimization, estimation, and control by Arthur E. Bryson, Yu-Chi Ho - Book (PDF File)
    • Optimal control and estimation by Stengel, Robert F - Book (PDF File) and Some Notes from the Book
